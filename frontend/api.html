Space Alone API Documentation

Complete API reference for the Space Alone platform

Overview

Base URL: http://localhost:3000/api

All responses are in JSON format. API uses JWT for authentication where required.

API Version: 1.0.0
Rate Limits:
‚Ä¢ General: 100 requests per 15 minutes
‚Ä¢ Auth: 5 requests per 15 minutes
‚Ä¢ NASA: 30 requests per minute

Authentication

Protected endpoints require JWT token in Authorization header:

Authorization: Bearer YOUR_JWT_TOKEN

Obtain token via /api/auth/login endpoint.

Authentication Endpoints

POST /api/auth/login

Admin login - Returns JWT token

Request Body:
Parameter Type Required Description
username string Yes Admin username
password string Yes Admin password
Example Request:
fetch('/api/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    username: 'admin',
    password: 'your_password'
  })
});
Response Example:
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": 1,
    "username": "admin"
  }
}
POST /api/auth/register

Register new admin user

Error Codes

Code Description
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid or missing token
403 Forbidden - Admin access required
404 Not Found - Resource doesn't exist
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error
