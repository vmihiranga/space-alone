Space Alone API Documentation

Complete API reference for the Space Alone platform

Overview

Base URL: http://localhost:5000/api

All responses are in JSON format. API uses session-based authentication (no JWT).

API Version: 1.0.0
Auth Method: Session-based (No JWT)
Rate Limits:
‚Ä¢ General: 10,000 requests per 15 minutes
‚Ä¢ Auth: 10 requests per 15 minutes
‚Ä¢ NASA & SpaceX: 30 requests per minute

Authentication

Protected endpoints use session-based authentication. After logging in, a session cookie is set automatically.

Login via /api/auth/login endpoint to establish a session.

Sessions are valid for 24 hours and stored in SQLite.

Authentication Endpoints

POST /api/auth/login

Admin login - Creates a session

Request Body:
Parameter Type Required Description
username string Yes Admin username
password string Yes Admin password
POST /api/auth/logout

Logout and destroy session (requires authentication)

GET /api/auth/verify

Verify current session status (requires authentication)

POST /api/auth/change-password

Change user password (requires authentication)

Request Body:
Parameter Type Required Description
currentPassword string Yes Current password
newPassword string Yes New password (min 6 chars)

Blog Posts Endpoints

GET /api/posts

Get all blog posts with pagination

Query Parameters:
Parameter Type Default Description
page integer 1 Page number
limit integer 100 Posts per page
GET /api/posts/search/query

Search posts by title or content

Query Parameters:
Parameter Type Required Description
q string Yes Search query
GET /api/posts/:id

Get single post by ID or slug

POST /api/posts

Create new blog post (requires admin authentication)

Request Body:
Parameter Type Required Description
title string Yes Post title
content string Yes Post content
image_url string No Featured image URL
PUT /api/posts/:id

Update existing post (requires admin authentication)

DELETE /api/posts/:id

Delete post (requires admin authentication)

GET /api/posts/:id/likes

Get like count for a post

POST /api/posts/:id/like

Like a post (uses session or IP for identification)

DELETE /api/posts/:id/like

Unlike a post

GET /api/posts/:id/liked

Check if current user liked a post

POST /api/posts/:id/share

Track post share

Request Body:
Parameter Type Required Description
platform string Yes Social platform (twitter, facebook, etc.)
GET /api/posts/:id/shares

Get share count for a post

Solar System Endpoints

GET /api/solar-config

Get all solar system configurations

POST /api/solar-config

Create solar system configuration (requires admin authentication)

Request Body:
Parameter Type Required Description
planet_name string Yes Planet name
size number No Planet size (default: 1)
distance number No Distance from sun (default: 1)
color string No Planet color (default: #ffffff)
speed number No Orbit speed (default: 1)
info string No Planet information
PUT /api/solar-config/:id

Update solar system configuration (requires admin authentication)

DELETE /api/solar-config/:id

Delete solar system configuration (requires admin authentication)

Error Codes

Code Description
400 Bad Request - Invalid parameters
401 Unauthorized - Session expired or invalid
403 Forbidden - Admin access required
404 Not Found - Resource doesn't exist
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error
Error Response Format:
{
  "error": "Route not found",
  "path": "/api/invalid",
  "method": "GET",
  "timestamp": "2024-10-20T12:00:00.000Z",
  "suggestion": "Check /api for available endpoints"
}
